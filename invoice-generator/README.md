# Local Invoice Generator

This app runs locally in your browser and lets you create invoice PDFs using your computer's print-to-PDF feature.

## Run it

From this folder, start a local server:

```bash
python3 -m http.server 3000
```

Then open http://127.0.0.1:3000 in your browser.

## Features

- Enter your business details once and reuse them for every invoice
- Add client details and line items
- Preview the invoice live
- Generate a PDF using the browser print dialog
